## Requirements
- zsh

- main
  - cargo

  - starship
  - eza
- option
  - Homebrew
  - Rye

## Installation

```bash
bash setup.sh -i
```

- `-i` Install & symlink dotfiles
- `-s` Symlink dotfiles
- `-e` Install extra packages (Homebrew)
